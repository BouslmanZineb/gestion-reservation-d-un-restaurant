
# site de gestion de réservations - Restaurant

##  Description
Ce projet est une application web permettant de gérer les réservations d’un restaurant.  
Elle permet de centraliser les informations clients, organiser les réservations et optimiser le suivi de l’activité.

---

##  Objectifs du projet
- Concevoir une application web complète (front-end + back-end)
- Mettre en place une base de données relationnelle
- Structurer et exploiter des données de manière efficace
- Développer une logique métier pour la gestion des réservations

---

##  Technologies utilisées
- PHP (back-end)
- MySQL (base de données)
- HTML / CSS / JavaScript (front-end)

---

##  Fonctionnalités principales
- Création, modification et suppression des réservations
- Gestion des informations clients
- Interface utilisateur dynamique et intuitive
- Stockage et manipulation des données via SQL

---

##  Base de données
- Modélisation relationnelle (tables clients, réservations…)
- Utilisation de requêtes SQL pour la gestion des données
- Structuration des données pour une exploitation optimale
